Серёга вывел формулу - смещение фокуса равно -2(F/D)^2 dH, где dH - мода цернике в микронах.
F/D у нас 8, для перевода dH в микроны надо взять эффективную длину волны изображения в любых единицах. В них и получишь смещение фокальной плоскости относительно детектора.
То есть для твоего значения -2 смещение в полосе V получается равно +0.128 мм.
